## 根据日期参考


> 这是一个部署在 Cloudflare Workers 上的高性能、高安全性的青龙面板管理机器人。它不仅是一个通知工具，更是一个完整的移动端管理终端，旨在通过 Telegram 彻底替代繁琐的网页端操作。

✨ 核心特性
 * ☁️ 零成本部署：基于 Cloudflare Workers，无需服务器，完全运行在免费额度内。
 * 🔐 金融级安全：
   * AES-256-GCM 加密：用户凭证加密存储，隐私安全。
   * 无痕 PIN 码：加密密钥由用户 PIN 码动态派生，服务器不存储 PIN 码，实现真正的零信任架构。
 * 👥 多用户支持：支持多用户独立绑定面板，具备完善的管理员权限管理（拉黑/禁言/删除）。
 * 📂 脚本一键分发：支持直接发送 .js/.py/.sh 文件或 GitHub/Gitee 链接，自动上传并创建任务。
 * ⚡ 全功能覆盖：
   * 任务管理：分页列表、日志查看、定时修改、运行/停止。
   * 变量/依赖/订阅：支持环境变量增删改查、系统依赖安装、仓库订阅同步。
 * 🔌 扩展能力：内置标准的 REST API 接口，方便外部程序集成。
🚀 快速开始
 * Fork 本仓库并部署到 Cloudflare Workers。
 * 配置变量：在 KV 中创建 USER_CONFIGS，并在环境变量中设置 TG_BOT_TOKEN 和 ADMIN_USER_IDS。
 * 初始化：访问 /set-webhook 激活机器人。
